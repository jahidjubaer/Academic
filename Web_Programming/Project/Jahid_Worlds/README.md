# Jahid Worlds Project  

Welcome to the **Jahid Worlds** project repository! This application is designed for smart shopping, where customers can browse a menu and make purchases seamlessly, much like a POS (Point of Sale) system.  

---

## Technology Stack  

### Front-End  
- **Frameworks/Libraries**: Next.js, React.js, and Tailwind CSS.  
- **Features**: Modern, responsive design with a focus on ease of use for shopping.  

### Back-End  
- **Technologies**: Next.js (API Routes for server-side logic).  
- **Features**: Core business logic and CRUD operations for product and order management.  

### Database  
- **Type**: MongoDB (NoSQL Database).  
- **Features**: Scalable and optimized for storing product and order data.  

### Payment Integration  
- **Provider**: Stripe  
- **Features**: Secure and fast payment processing for customers.  

---

## Features  

- **Dynamic Menu**:  
  - A visually appealing menu for products.  
  - Includes product details like descriptions, prices, and categories.  

- **Friendly Design**:  
  - Optimized for in-store use with responsive components.  

- **Order Management**:  
  - Add items to a cart.  
  - Calculate totals and handle orders efficiently.  

- **Payment System**:  
  - Integrated with Stripe for smooth transaction processing.  

- **Additional Features**:  
  - POS-like interface for easy navigation.  
  - Real-time updates for order summaries.  

---

## Getting Started  

### Prerequisites  

Ensure the following tools are installed:  
- Node.js
- MongoDB (Local or cloud-based).  

## Installation

### Clone the repository:  

```bash
git clone https://github.com/jahidjubaer/Academic/tree/master/Web_Programming/Project/Jahid_Worlds
```


### Navigate to the project directory:

```bash
cd Jahid_Worlds
```


### Install dependencies:

```bash
npm install
```

### Set up environment variables:

Create a .env.local file in the root directory.
Add the following variables:

```
MONGODB_URI=your_mongodb_connection_string
STRIPE_SECRET_KEY=your_stripe_secret_key
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=your_stripe_publishable_key
```

### Start the development server:
```bash
npm run dev
```

Access the application at http://localhost:3000.
