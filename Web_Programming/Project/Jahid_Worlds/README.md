# Jahid Worlds Project  

Welcome to the **Jahid Worlds** project repository! This application is designed for offline smart shopping, where customers can browse a menu and make purchases seamlessly, much like a POS (Point of Sale) system.  

---

## Technology Stack  

### Front-End  
- **Frameworks/Libraries**: Next.js, React.js, and Tailwind CSS.  
- **Features**: Modern, responsive design with a focus on ease of use for offline shopping.  

### Back-End  
- **Technologies**: Next.js (API Routes for server-side logic).  
- **Features**: Core business logic and CRUD operations for product and order management.  

### Database  
- **Type**: MongoDB (NoSQL Database).  
- **Features**: Scalable and optimized for storing product and order data.  

### Payment Integration  
- **Provider**: Stripe  
- **Features**: Secure and fast payment processing for customers.  

---

## Features  

- **Dynamic Menu**:  
  - A visually appealing menu for products.  
  - Includes product details like descriptions, prices, and categories.  

- **Offline-Friendly Design**:  
  - Optimized for in-store use with responsive components.  

- **Order Management**:  
  - Add items to a cart.  
  - Calculate totals and handle orders efficiently.  

- **Payment System**:  
  - Integrated with Stripe for smooth transaction processing.  

- **Additional Features**:  
  - POS-like interface for easy navigation.  
  - Real-time updates for order summaries.
